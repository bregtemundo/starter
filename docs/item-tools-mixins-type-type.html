1.2.6.3 type($sizemap, $leader, $trailer, $font)

Mixin for setting font size (and margins)
Output depends on $baseline variable (settins/_variables.scss):

  • if baseline => output with the megatype library to keep fonts to baseline grid
  • else we output font-size and line-height
$fontsizes are set in settings/_variables.scss:
(base: (null: 16px, 1.58, lg: 18px), h1: (null: 40px, 1.54, lg: 100px, 1), h2: (null: 32px, 1.15), h3: (null: 32px), h4: (null: 30px), h5: (null: 27px), h6: (null: 25px))

  • the key of the $fontsizes map

    $sizemap

  • the vr rithm above, can be unitless or static value

    $leader

  • the vr below

    $trailer

  • the name of the font-face (body, title)

    $font
